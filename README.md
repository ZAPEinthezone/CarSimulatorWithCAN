# CarSimulatorWithCAN

A car simulation project with CAN bus communication.

## Features
- CAN message simulation
- Vehicle behavior modeling
- Modular project structure

## How to Run
1. Clone the repo  
   `git clone https://github.com/ZAPEinthezone/CarSimulatorWithCAN.git`
2. Open the project
3. Build and run
